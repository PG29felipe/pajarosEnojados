PAJAROS ENOJADOS (Angry birds in spanish) by PG29Felipe

git repo:

https://github.com/PG29felipe/pajarosEnojados

this Project contains both the level editor ad the actual game.

to run the level editor:
1. open the CMD in the "PajarosEnojadosLevelEditor" folder.
2. run this comand: npm run dev.
3. it should run a local server.
4. go to LevelEditor+Game\PajarosEnojadosLevelEditor\public and double click index.html, this should open the level editor.

to run the actual game.
1. open the CMD in the "PajarosEnojadosGame" folder
2. run thi command: npx http-server -c-1
3. this should start an http server, it is usually available in http://127.0.0.1:8080/  

to play the game:

1.enter the name of the level in the "level ID" fiel and click "Load".
2. if a level with that name exists, it loads and you can start playing.
3. click the top of the catapult (ORANGE BLOCK) to spawn and launch a bird
4. you have to kill the pigs (green spheres)
5. you only have 3 oportunities.
